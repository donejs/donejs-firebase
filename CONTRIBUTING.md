## Developing Locally

### Forking & cloning the repository

```shell
git clone git@github.com:donejs/donejs-firebase.git
cd donejs-firebase
```

### Installing the dependencies

```shell
npm install
```

### Running the tests

You can run the tests with:

```shell
npm test
```
